# Sons of Unity 🌍

**Sons of Unity** is a global movement and digital platform built to unite people through brotherhood, purpose, service, and growth.  
The project aims to create a strong worldwide community powered by technology, transparency, and shared values.

🌐 Live Website: https://sonsofunity.com  
🚀 Hosted on: Cloudflare Pages  
📦 Source Control: GitHub

## 🚀 Deployment

This project uses **Cloudflare Pages with GitHub integration**.

### How it works:
1. Code is pushed to GitHub  
2. Cloudflare Pages automatically builds & deploys  
3. Website updates go live globally within seconds  

No manual servers. No maintenance. Fully scalable.
