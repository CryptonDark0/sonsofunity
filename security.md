# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in **Sons of Unity**, we ask that you report it responsibly.

- **Email:** security@sonsofunity.com  
- **GitHub Issues:** [Report via Issues](https://github.com/CryptonDark0/sonsofunity/issues)  
- **Security.txt:** [https://sonsofunity.com/.well-known/security.txt](https://sonsofunity.com/.well-known/security.txt)

Please include as much detail as possible in your report (steps to reproduce, affected components, potential impact). This helps us investigate and resolve issues quickly.

---

## Supported Versions

We currently support security updates for the following branches:

| Version/Branch | Supported |
|----------------|------------|
| `main`         | ✅ Active support |
| Other branches | ❌ Not supported |

---

## Disclosure Policy

- We will acknowledge receipt of your report within **72 hours**.  
- We will provide an initial assessment within **7 days**.  
- Valid vulnerabilities will be patched as quickly as possible.  
- We will publicly disclose the issue only after a fix has been released.  

---

## Acknowledgments

We thank all security researchers and contributors who help keep **Sons of Unity** safe. Responsible disclosures may be recognized in our [Acknowledgments section](https://github.com/CryptonDark0/sonsofunity/security/policy#acknowledgments).
